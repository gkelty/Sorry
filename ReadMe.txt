To run the game run the gameloop.py file from the command line or python interpreter.
In order to run the gameloop you need to be on the UVM network because of our Database.
To get to main menu enter a username to either create a new user or use an existing one.
From the main menu select any option, the resume game is not implemented yet.
To start new game click the new game and set up your settings using text
To make a computer move you need to click the pink square although you cannot chose where they go or what pawn gets moved
To Move your pawn first select the pawn you want to move(Highlighted in pink)
Then select where you would like to move it(another highlighted tile)
Game instructions can be found in the instructions tab of the main menu
The seven card only moves forward seven, you cannot split
